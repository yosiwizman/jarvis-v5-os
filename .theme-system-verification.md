### V5 Theme System (Jarvis V5)
**Locations**:
- Context: `apps/web/src/context/ThemeContext.tsx`
- CSS Variables: `apps/web/app/globals.css` (lines 31-87)
- Settings UI: `apps/web/app/settings/page.tsx` (lines 320-450)

**Architecture**:
1. **React Context Provider** (`ThemeContext`)
   - Strongly typed with TypeScript
   - Manages both `mode` (light/dark) and `colorTheme`
   - Instant updates via state management
   - Persists to localStorage (`jarvis-v5-theme`)

2. **CSS Variables** (RGB format for alpha compositing)
   ```css
   .theme-cyber-blue {
     --jarvis-accent: 59, 130, 246;
     --jarvis-accent-muted: 37, 99, 235;
     --jarvis-bg-grid: 59, 130, 246;
     --jarvis-panel-surface: 30, 58, 138;
     --jarvis-glow: 59, 130, 246;
   }
   ```

3. **Settings UI Integration**
   - 5 theme buttons with visual previews
   - Color dots matching theme
   - Active state with glow effects
   - Responsive grid layout

4. **HTML Class Application**
   - Light/dark mode: `<html class="light">` or `<html class="dark">`
   - Color theme: `<html class="theme-cyber-blue">`
   - Applied automatically via `useEffect` in `ThemeProvider`

## Enhancements Over V3

## V5 Usage Example

```typescript
import { useTheme } from '@/context/ThemeContext';

function MyComponent() {
  const { colorTheme, setColorTheme } = useTheme();
  
  return (
    <button onClick={() => setColorTheme('midnight-purple')}>
      Current: {colorTheme}
    </button>
  );
}
```
