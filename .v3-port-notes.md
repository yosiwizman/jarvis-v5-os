### HUD Widget Data (V3 approach)

- CPU: Static demo value (42%)

- Network: Static demo value (215 KB/s)
- "SYNCED" indicator: Always shows as synced with pulsing dot

### Theme Context (V3 structure)
- V3 uses `settingsService` with localStorage
- Themes apply as CSS inline styles via RGB values
- No dark/light mode - only color themes

### Settings Persistence (V3 approach)
- OpenAI key: localStorage only
- Spotify/HF: Server API endpoints (`/api/spotify/config`, `/api/huggingface/config`)
- Form saves trigger server POST requests
